## Features

* 時間割の表示・表示期間変更機能
* 定期的にITC-LMSの新着更新を取得する機能
* 講義詳細の表示機能
* 課題詳細の表示機能
* アプリ内でPDFを表示する機能
* 添付ファイルをダウンロードする機能
